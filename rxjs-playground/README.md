# Reaktive Programmierung mit RxJS

In diesem Workshop tauchen wir tief ein in die Welt der Reaktiven Programmierung mit RxJS. Es dreht sich alles um reaktive Datenströme, Observables, Operatoren, Subjects und Subscriptions.

### Webanwendung im Browser

```bash
cd rxjs-playground
npm install
npm start
```

Öffne den Browser unter der URL [http://localhost:**4300**](http://localhost:4300), um die Anwendung zu sehen.
Die Übungen befinden sich im Ordner `rxjs-playground/src/app/exercises/`.
